# 📋 Instruções para o Copilot - UTFPR Campus Calendar

## Antes de fazer mudanças importantes ⚠️
Sempre pergunta ao usuário ANTES de:
- Refatorar código existente
- Mover/renomear arquivos
- Deletar código
- Fazer grandes mudanças estruturais
- Alterar fluxo de autenticação/navegação

## Pode fazer direto ✅
- Corrigir erros óbvios e bugs
- Adicionar novos arquivos pequenos
- Completar funcionalidades incompletas (como login com Firebase)
- Sugerir melhorias (com confirmação)
- Atualizar imports necessários

## Padrão de resposta
1. Explicar O QUE vai fazer
2. Mostrar um resumo/preview se for grande
3. Perguntar "Quer que eu continue? (sim/não)"
4. Só depois executar

## Prioridades do Projeto (Entrega 18/06)
- ✅ Autenticação Firebase funcional
- ✅ Proteção de rotas (impedir acesso sem login)
- 🔄 Persistência de dados (Firestore)
- 🔄 Dados reais (não hardcoded)
- 🔄 Documentação + Tutorial

## Estrutura que queremos
```
styles/          ← CSS/StyleSheets
contexts/        ← Autenticação global
services/        ← Firebase
utils/           ← Constantes e helpers
components/      ← UI reutilizáveis
app/             ← Telas/páginas
```
