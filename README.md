# Agenda Acadêmica UTFPR

Aplicativo mobile em React Native para centralizar eventos, prazos acadêmicos e editais da universidade.  
  

---

## Telas
1. Login
2. Cadastramento
3. Tela inicial  
4. Calendário  
5. Detalhes do evento
6. Busca de eventos
7. Perfil
